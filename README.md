🛒 Product Management Application

📌 Project Overview

The Product Management Application is a full-stack web application designed to manage products efficiently.
It allows users to perform complete CRUD (Create, Read, Update, Delete) operations on products with a user-friendly interface.
This project demonstrates the integration of a backend built using Spring Boot and a frontend developed using React.js, connected via REST APIs with Cross-Origin support.

🚀 Features

* ➕ Add new products with price details
* 📋 View all available products
* ✏️ Update product information (e.g., modify price)
* ❌ Delete products
* 🔄 Seamless communication between frontend and backend using REST APIs
* 🌐 Cross-Origin Resource Sharing (CORS) enabled

⚙️ How It Works

1. The user interacts with the React frontend UI
2. Frontend sends HTTP requests to Spring Boot backend
3. Backend processes requests and performs CRUD operations
4. Data is returned and displayed dynamically on UI

▶️ How to Run the Project

Backend (Spring Boot)

1. Open project in Eclipse
2. Run the Spring Boot application
3. Ensure server is running on default port (e.g., 8080)

Frontend (React)

1. Open project in VS Code
2. Run:

   ```bash
   npm install
   npm start
3. Open browser at:

   http://localhost:3000
